Summary: Latitude vs. X

The purpose of this project was to analyze how weather chnage as you get closer to equater. To acomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude
